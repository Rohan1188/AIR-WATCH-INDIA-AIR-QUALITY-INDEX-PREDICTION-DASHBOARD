## 📌 Project Overview

**AIR WATCH INDIA** is an advanced **Air Quality Index (AQI) Prediction & Monitoring Dashboard** developed using **Machine Learning, Data Analytics, and Interactive Visualization** techniques.

The project predicts AQI levels across Indian cities using environmental and pollution parameters while providing an intuitive dashboard for real-time monitoring and analysis.

This platform helps:

* Citizens monitor pollution levels
* Researchers analyze environmental trends
* Authorities make data-driven decisions
* Students learn ML-based environmental prediction systems

Air pollution remains one of the biggest environmental challenges in India, and this project aims to contribute toward awareness and smart monitoring solutions. ([GitHub][1])

# 🧠 Machine Learning Models Used

The project experiments with multiple ML algorithms for accurate AQI prediction:

* Linear Regression
* Random Forest Regressor
* Decision Tree Regressor
* XGBoost Regressor
* Lasso Regression
* Artificial Neural Network (ANN)

These models are evaluated using:

* MAE
* RMSE
* R² Score

AQI prediction systems commonly use regression-based ML approaches with pollutant datasets and meteorological parameters. ([GitHub][1])

# 🔧 Backend Setup

## Create Virtual Environment

```bash
python -m venv venv
```

## Activate Environment

### Windows

```bash
venv\Scripts\activate
```

### Linux / Mac

```bash
source venv/bin/activate
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Start Backend

```bash
python app.py
```

# 📌 AQI Categories

| AQI Range | Category     | Impact                     |
| --------- | ------------ | -------------------------- |
| 0–50      | Good         | Minimal Impact             |
| 51–100    | Satisfactory | Minor Breathing Discomfort |
| 101–200   | Moderate     | Breathing Issues           |
| 201–300   | Poor         | Health Effects             |
| 301–400   | Very Poor    | Respiratory Illness        |
| 401–500   | Severe       | Serious Health Risk        |
